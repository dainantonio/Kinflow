# Contributing

Thanks for contributing to Orbit.

## Development
1. Fork and clone the repo.
2. Install dependencies: `npm install`.
3. Copy env template: `cp .env.example .env` and fill required values.
4. Validate setup: `npm run setup:check`.
5. Start dev server: `npm run dev`.

## Pull Requests
- Keep changes focused and mobile-first.
- Ensure no dead-end taps or clipped UI in app flows.
- Run `npm run build` before opening PR.
- Add screenshots for visual UI changes.

## Code style
- Follow existing React/Tailwind conventions.
- Prefer small reusable UI primitives.
